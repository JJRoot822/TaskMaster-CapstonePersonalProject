PC & Web Capstone Project Documentation


Application's purpose

This application is intended to be used to help independent developers effectively manage development projects and various details surrounding them including tasks to complete, bugs to fix, issues to resolve and more. You will also be able to search through projects and their associated items.


App Platforms

  • macOS 14
  • iOS 17
  • iPadOS 17

Technologies to be Used

  • The Swift Programming Language
  • SwiftUI (Apple’s latest user interface framework)
  • ASP.NET Core Web API using .NET 7 for the backend
  • Entity Framework Core and SQLite for the database connectivity (Was originally going to be MySQL, but complications arose, so to avoid further issues, I chose to go with SQLite, but in future for release when I’m graduated it can easily use SQL Server in Azure or some other cloud service)
  • Username, email and password authentication, which will be handled by the web service

Changes Since Sprint 1

The primary changes that have taken place since the end of the previous sprint is that the API written in C# has been completed and tested and now works. I can now begin working on the iOS/iPadOS/macOS app written in Swift and SwiftUI. I have also cut down the number of features the app will have to ensure that I have enough time to actually finish the project at a satisfactory degree and as a result of undoing some of the overthinking and overcomplicating things I had done in the initial planning stage.
